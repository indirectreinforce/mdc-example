## 用于 Web 的 Material Design UI 组件 MDC Web 示例源代码.
使用方法 [使用 Material Design 的 UI 组件 MDC Web](https://unus.me/usemdui-mdc/)

```
git clone https://github.com/unusme/mdc-example.git
```
```
cd mdc-example 
```
```
npm install
```
```
npm start
```
查看效果： http://localhost:8080 

生产环境构建 
```
npm build 
```
查看 `dist` 目录
- bundle.css
- bundle.js
